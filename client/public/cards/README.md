# Card Images

Place your role card PNG images here. The filenames must match exactly:

| File           | Role     |
|----------------|----------|
| mafia.png      | Mafia    |
| doctor.png     | Doctor   |
| police.png     | Police   |
| spy.png        | Spy      |
| villager.png   | Villager |
| reporter.png   | Reporter |
| soldier.png    | Soldier  |
| lover.png      | Lover    |

Recommended card size: **400 × 560 px** (portrait, 5:7 ratio).

If an image is missing, the card back will fall back to showing
the role's emoji icon and name with a matching glow color.
