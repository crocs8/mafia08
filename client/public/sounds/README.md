# 🔊 Mafia08 Game Sounds

Place your audio files in this folder with these **exact** filenames:

| File | Format | When it plays |
|------|--------|--------------|
| `night-music.wav` | WAV | Loops softly in the background when Night phase begins |
| `morning-bell.wav` | WAV | Plays once (3s) when Day phase begins |
| `kill-sound.mp3` | MP3 | Plays simultaneously with the morning bell if someone was killed during the night |
| `card-flip.mp3` | MP3 | Short flip/whoosh sound when the role card is tapped and flips |

These files are already wired up in code — just drop them here and they'll work.
