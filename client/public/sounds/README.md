# 🔊 Mafia08 Game Sounds

Place your `.mp3` files in this folder with these **exact** filenames:

| File | When it plays |
|------|--------------|
| `night-music.mp3` | Loops softly in the background when Night phase begins |
| `morning-bell.mp3` | Plays once (3s) when Day phase begins |
| `kill-sound.mp3` | Plays simultaneously with the morning bell if someone was killed during the night |

These files are already wired up in code — just drop them here and they'll work.
